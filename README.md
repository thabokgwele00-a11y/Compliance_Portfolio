🚀 Technical Portfolio Ecosystem
The projects below are designed to simulate real-world data scenarios, financial crime typologies, and compliance mandates within the South African banking and financial sector.

🔍 Financial Crime & Forensic Analytics
FICA Section 28 AML Pipeline: An automated anti-money laundering triage engine. It runs advanced BigQuery window functions to catch split-deposit structuring (bad actors trying to game the R24,999.99 Cash Threshold Report boundary across time intervals).

Forensic Fraud Velocity Engine: A high-velocity transaction monitoring system built to flag complex fraud typologies—specifically detecting "CNPP Merchant Sweeps" and "Structured Smash-and-Grabs" by analyzing rapid channel alternation (POS vs. Online) within 15-minute windows.

Automated Fraud Reporter: A Python automation and schema synchronization pipeline using the Google Cloud SDK. It executes forensic queries, isolates automated alert data, and manages daily report extraction for operational audit teams.

📊 Regulatory Reporting & Risk Modeling
BA 900 SARB Reporting Engine: A programmatic balance sheet mapping engine built to compile institutional macroprudential returns for the South African Reserve Bank. It includes automated ledger validation checks and handles data anomalies safely.

IFRS 9 / GRAP 104 Credit Risk Engine: A technical wholesale credit risk calculator that segments banking portfolios into impairment stages based on credit deterioration triggers, calculating Expected Credit Losses programmatically.

Predictive Credit Risk Scorer: A machine learning model integrated directly into the data warehouse using BigQuery ML (BQML). It builds logistic regression algorithms to score campaign risk on historical customer records.
